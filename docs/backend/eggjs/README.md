## 安装

推荐直接使用脚手架，只需几条简单指令，即可快速生成项目（`npm >=6.1.0`）

```sh
mkdir egg-project && cd egg-project
npm init egg --type=simple
npm i
```

## 启动项目

```sh
npm run dev
```
